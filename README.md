# Local Attributes

[Website/Releases](http://www.polydes.com/repo/engine/com.polydes.localattrs) |
[Issues](https://github.com/polydes/local-attrs/issues)

</div>

This [Stencyl](https://www.stencyl.com/) extension allows attributes to be defined within an event, for cleaner code.

## Using this Stencyl extension

This is a free extension. Installation instructions and guides can be found [here](http://www.polydes.com/repo/engine/com.polydes.localattrs/).
